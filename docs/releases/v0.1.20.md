# v0.1.20

Release highlights:

- Added an end-to-end validation stack:
  - `iints doctor`
  - `iints validation-profiles`
  - `iints validate-run`
  - `iints contract-verify`
  - `iints certify-run`
  - `iints scorecard`
- Expanded research tooling:
  - leakage/split auditor (`iints research audit-split`)
  - calibration-first forecast evaluator (`iints research evaluate-forecast`)
  - Torch/ONNX parity checks (`iints research parity-check`)
- Dual-Guard hardening:
  - predictor uncertainty + OOD gates in the simulator
  - richer predictor/safety telemetry in run outputs
  - InputValidator fail-soft telemetry (`input_validator_fail_soft` and summary counters)
- Added structured demos in `examples/demos/`, including an Open-Logic architecture showcase.
- Added OhioT1DM v2 predictor config with band-weighted loss, early stopping, and meal preannounce feature support.

See `CHANGELOG.md` for the complete technical list.
